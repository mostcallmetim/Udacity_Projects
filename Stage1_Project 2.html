How The Web Works

The web or internet is a series of computers that communicate with each other through servers. When you go to a www.google.com for example your browser requests that web page's HTML document from the server and then displays it on your screen. Inline Vs. Block

Finally HTML elements are either inline or block. Block differs itself from inline in that block elements form an "invisible block" in which to display content. In the previous video you already may remembering hearing about some these elements known as divs or spans.

Lesson 2 Concepts

Developer Tools

Most web browsers have a way to view the code the webpage was written in. For example, in Gooogle, the Chrome Developer Tools (DevTools for short), are provide for web developers for deep access into the internals of the browser and other web applications.

DOM

The Document Object Model (DOM) is a programming interface for HTML documents. It provides a way that the structure can be accessed from programs so that they can change the style and content. The DOM is used by browser to translate HTML tags in tree-like structure.

CSS

Cascading Style Sheets (CSS) is a language used for the styling of a webpage. CSS works in conjunction with HTML forming divisions and layouts of the page. Looking closely you could break down any webpage into sections or boxes.

CSS Classes and ID

In CSS, a class selector name preceded by a period [.] and an ID selector is a name preceded by a hash character [#]. You can use class and id selectors to modify mutiple HTML tags of same name rather than modifying them individually.

CSS divs and spans

The difference between <span> and <div> is that a span element is in-line, such as inside a paragraph. Whereas a div (division) element is block-line and used to group larger chunks of code. each div forms a block around the code allowing it to modified.
